
Assignment 7 : 
This assignment is to write complete "Web 2.0" application that includes user login sessions, a PHP web service with AJAX and graphical effects from the Scriptaculous JavaScript library             

Tasks: 
Build a online to-do list site called "Remember the Cow". It should includes following features:
- user login sessions
- after logging in, the user can manipulate his/her to-do list [add an item, delete an item and drag the items into different order]
- any changes made to the list will be saved to the web server using AJAX
- if user leaves the page and returns later, the current state of the list will be remembered
